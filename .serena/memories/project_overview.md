## 项目概述
- 这是一个 1v1 回合制对战模拟器，角色拥有生命/攻击/防御/速度等浮点属性，并通过状态、被动、主动技能互动；速度决定出手顺序，生命≤0 即判负。
- 代码使用 Python 3（pyproject 要求 >=3.13），依赖通过 uv 管理；当前没有第三方依赖，重点在自定义战斗框架。
- 主要结构：`main.py` 负责选择参赛角色、执行循环赛并打印示例对局；`bh3_duel_sim/` 为核心包，包含 `characters/`（基类与具体女武神/占位角色）、`simulator.py`（回合驱动＋统计）、`stats.py`（属性 dataclass）、`logger.py`（开关式日志器）。
- `characters/base.py` 统一管理随机数、主动技能冷却、状态回合处理等通用机制，子类只需覆盖状态效果、被动和主动逻辑即可。
- 目前主程序默认只跑科拉莉 vs 布洛妮娅，若要扩展可通过 `build_valkyrie_roster()` 选择其它角色或新增工厂。
